# Goed

Goed is a simplified tool to check and assign files with gpg keys ed25519


```
Use it to generate, sign and check ed25519 gpg keys

Usage:
  goed [command]

Available Commands:
  check       Check the file with the file signature and public key
  completion  Generate the autocompletion script for the specified shell
  generate    Generate 2 keys files
  help        Help about any command
  seed        Generate a random seed, retuns it encoded on base64
  sign        Sign a file generating a signature file

Flags:
  -h, --help   help for goed

Use "goed [command] --help" for more information about a command.
```
